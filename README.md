# SYwork 工时记录系统

## 项目简介

SYwork 是一款专为个人设计的工时记录微信小程序，旨在帮助用户高效地规划、跟踪和管理工作时间。通过简洁直观的界面和实用的功能，用户可以轻松制定工作计划、安排日常排班、统计工时数据，并进行个人数据管理。

## 核心功能

### 📋 计划页面 (Plan)
- 创建和管理每日工作任务
- 设置任务优先级和预计工时
- 查看任务完成状态

### 📅 排班页面 (Schedule)
- 制定周/月度工作排班表
- 可视化展示工作时间安排
- 支持拖拽调整任务时间

### 📊 统计页面 (Statistics)
- 图表化展示工时统计数据
- 分析工作效率和时间分配
- 导出统计报表

### 👤 我的页面 (Profile)
- 个人信息管理
- 数据导出和导入（JSON格式）
- 缓存清理功能

## 技术特性

### 离线使用
- 所有数据通过 `wx.setStorageSync` / `wx.getStorageSync` 存储在本地
- 无需网络连接即可使用全部功能

### 免登录使用
- 登录状态检查为可选功能
- 即使未登录也可使用所有核心功能
- 登录功能为框架预留，暂未完整实现

## 使用说明

### 启动方式
1. 直接在微信开发者工具中打开小程序
2. 可跳过登录步骤直接使用所有功能
3. 所有数据将保存在当前设备中

### 注意事项
- 清除小程序缓存会导致数据丢失
- 建议定期导出重要数据作为备份
- 不同设备间的数据无法同步

## 数据管理

### 数据导出
- JSON格式：完整的数据结构，适合备份和迁移
- 导出内容包含：
  - 自定义班次模板数据（包括排列顺序）
  - 排班数据
  - 统计数据（根据排班实时计算）

### 数据导入
- 支持从JSON文件导入数据
- 导入时会将各个数据恢复至对应板块：
  - 自定义班次模板板块
  - 排班板块
- 统计数据会根据导入的排班数据实时更新
