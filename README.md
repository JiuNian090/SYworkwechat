# SYwork|工时记录系统

## 项目介绍
SYwork|工时记录系统是一个基于微信小程序的工时记录和管理系统，帮助用户记录日常工作时间、生成统计报表和导出数据。

## 功能特性
- 工作计划管理
- 排班记录
- 数据统计分析
- 数据导出（CSV/Excel格式）
- 用户个人信息管理

## 技术栈
- 微信小程序原生开发
- JavaScript
- WXML/WXSS

## 项目结构

## 开发环境配置

### 1. 微信开发者工具
- 下载并安装微信开发者工具
- 使用小程序AppID创建项目

### 2. 服务器域名配置
1. 登录微信公众平台
2. 进入"开发管理"->"开发设置"
3. 在"服务器域名"部分配置以下域名：
   - request合法域名（必须是HTTPS协议）
   - socket合法域名
   - uploadFile合法域名
   - downloadFile合法域名

### 3. API配置
1. 配置API密钥和基础URL：
   - 在`app.js`的`globalData.config`中配置API密钥和基础URL
   - 将`your_api_key_here`替换为实际的API密钥
   - 将`https://your-api-domain.com/api`替换为实际的API基础URL

2. 在`config.js`中配置API路径和参数

### 3. 文件系统权限配置
已在`app.json`中配置了文件系统读写权限：
```json
"permission": {
  "filescope.read": {
    "desc": "用于读取小程序内的文件"
  },
  "filescope.write": {
    "desc": "用于写入小程序内的文件"
  }
}
```

## 部署流程
1. 在微信公众平台完成小程序基本信息配置
2. 配置服务器域名
3. 使用微信开发者工具上传代码
4. 在微信公众平台提交审核
5. 审核通过后发布上线

## 数据导出功能
本系统支持将工时记录数据导出为CSV格式文件，用户可以在"个人中心"页面点击"导出数据"按钮进行操作。

## 注意事项
1. 所有敏感信息（如API密钥）应通过环境变量配置
2. 文件导出功能需要用户授权文件系统读写权限
3. 网络请求需要在微信公众平台配置对应的服务器域名
