# 数据管理API文档

## 概述

本文档详细说明了SYwork工时记录系统中数据导出、导入和WebDAV备份功能的具体规定和实现方式。

## 数据导出功能

### 功能描述
数据导出功能允许用户将系统中的所有重要数据整合成一个ZIP文件，便于备份或迁移。

### 导出数据结构

系统会生成包含以下内容的ZIP文件：

1. **班次模板数据** (`班次模板.json`)
   - 包含用户创建的所有班次模板
   - 保留班次模板的排列顺序
   - 包含班次名称、时间、工时、类型和颜色等完整信息

2. **排班数据** (`排班数据.json`)
   - 包含用户所有的排班记录
   - 以日期为键，班次信息为值的键值对结构
   - 包含每日排班的完整信息
   - 包含统计数据

3. **图片文件** (`images/`文件夹)
   - 包含所有与排班相关的图片
   - 按周组织存储

### 导出内容说明

1. **自定义班次模板数据**
   - 导出为`班次模板.json`文件
   - 包含完整的班次模板信息
   - 添加时间戳用于版本管理

2. **排班数据**
   - 导出为`排班数据.json`文件
   - 包含所有排班记录
   - 自动计算统计数据
   - 添加时间戳用于版本管理

3. **图片文件**
   - 导出到`images/`文件夹
   - 保留原始图片质量
   - 按原始命名方式存储

## 数据导入功能

### 功能描述
数据导入功能允许用户从ZIP文件中恢复数据，将各个数据恢复至对应的板块。

### 导入数据处理流程

1. **数据验证**
   - 验证ZIP文件格式是否正确
   - 检查必需的文件结构是否存在

2. **数据恢复**
   - 解压ZIP文件
   - 将班次模板数据恢复至自定义班次模板板块
   - 将排班数据恢复至排班板块
   - 将图片文件恢复至对应位置

3. **数据刷新**
   - 刷新所有相关页面的数据展示
   - 重新计算并更新统计数据

### 导入后数据处理

1. **自定义班次模板板块**
   - 恢复所有班次模板
   - 保持原有排列顺序
   - 更新计划页面的班次模板列表

2. **排班板块**
   - 恢复所有排班数据
   - 更新排班页面的排班展示
   - 重新生成周/月视图

3. **图片文件**
   - 恢复所有图片文件
   - 保持原始文件结构
   - 更新相关页面的图片展示

4. **统计数据**
   - 根据导入的排班数据实时更新
   - 重新计算总工时、排班天数等统计信息
   - 更新统计页面的图表展示

## WebDAV备份功能

### 功能描述
WebDAV备份功能允许用户将数据自动备份到WebDAV服务器，实现数据的云端存储和多设备同步。

### 备份流程

1. **配置WebDAV**
   - 进入"我的"页面
   - 点击"配置WebDAV"按钮
   - 输入WebDAV服务器地址、用户名和密码
   - 可选：设置自定义备份文件夹

2. **执行备份**
   - 点击"备份到WebDAV"按钮
   - 系统自动生成包含所有数据的ZIP文件
   - 上传ZIP文件到WebDAV服务器
   - 自动清理旧的备份文件，只保留最新版本

3. **从WebDAV恢复**
   - 点击"从WebDAV恢复"按钮
   - 系统获取服务器上的备份文件列表
   - 选择要恢复的备份文件
   - 系统自动下载、解压并恢复数据

### WebDAV配置要求

1. **服务器地址**
   - 必须包含完整的协议（http://或https://）
   - 支持标准WebDAV协议的服务器
   - 推荐使用支持HTTPS的服务器

2. **权限要求**
   - 服务器必须允许PUT操作以上传文件
   - 部分服务器可能需要手动创建备份文件夹

3. **存储空间**
   - 确保服务器有足够的存储空间
   - 备份文件大小取决于数据量和图片数量

## 注意事项

1. **数据完整性**
   - 导出的ZIP文件包含所有必要数据
   - 导入时会验证数据完整性

2. **数据同步**
   - 导入数据后会自动刷新所有相关页面
   - 统计数据会根据排班数据实时更新

3. **兼容性**
   - 导出的ZIP文件可在不同设备间使用
   - 导入功能兼容历史版本的数据格式

4. **WebDAV备份**
   - 支持主流WebDAV服务器
   - 自动管理备份版本，只保留最新版本
   - 建议定期执行备份以确保数据安全

## 使用示例

### 导出数据示例

1. 进入"我的"页面
2. 点击"导出数据"按钮
3. 选择要导出的数据类型
4. 输入文件名（可选）
5. 系统生成ZIP文件
6. 点击"分享给好友"将文件发送给他人

### 导入数据示例

1. 进入"我的"页面
2. 点击"导入数据"按钮
3. 选择之前导出的ZIP文件
4. 系统自动解压并恢复数据
5. 所有页面数据自动刷新

### WebDAV备份示例

1. 进入"我的"页面
2. 点击"配置WebDAV"按钮
3. 输入WebDAV服务器信息
4. 点击"备份到WebDAV"按钮
5. 系统自动完成备份过程
6. 显示备份成功提示

### WebDAV恢复示例

2. 点击"从WebDAV恢复"按钮
3. 选择要恢复的备份文件
4. 系统自动完成恢复过程
5. 所有页面数据自动刷新
